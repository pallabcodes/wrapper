# Observability and SLOs

## SLOs
1. Availability target per service is at or above ninety nine point nine percent monthly
2. p95 latency budget per endpoint documented and tracked
3. Error rate budget defined as a fraction of total requests

## Alerts
1. Page on sustained SLO burn above policy
2. Ticket on single burn events that recover
3. Noise limits enforced on alert rules

## Dashboards
1. Standard dashboards for throughput and latency and errors
2. Tracing views for critical paths
3. Log views with request id and user id correlation
